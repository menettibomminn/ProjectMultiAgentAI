### Architecture

All agents follow the 10-step pipeline from the Agents/sheets/ reference:
1. Locate task in inbox
2. Parse and validate via JSON Schema + semantic checks
3. Idempotency check (skip duplicates)
4. Acquire resource lock (portalocker)
5. Generate structured report
6. Atomic write (tmp + rename)
7. Archive processed task
8. Write audit log with SHA-256 checksums
9. Update HEALTH.md
10. Release lock
